# Research Reference

## arXiv

```bash
noman arxiv search "machine learning"
noman arxiv get --id 2301.12345
noman arxiv download --id 2301.12345 --dest /tmp/paper.pdf
```

## Polymarket

```bash
noman polymarket search "election"
noman polymarket info <market_id>
```

## RSS

```bash
noman rss add "https://example.com/rss"
noman rss list
noman rss update
noman rss remove <feed_id>
```

## Blog Watcher

```bash
noman blog add "https://blog.example.com"
noman blog list
noman blog update
```

## LLM Wiki

```bash
noman llm-wiki search "transformer"
noman llm-wiki get <page_id>
```
