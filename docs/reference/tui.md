# TUI Reference

## Overview

NoMan uses Textual for its terminal UI with enhanced features.

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| Ctrl+C | Cancel current operation |
| Ctrl+E | Expand output |
| Ctrl+D | Show diff |
| Ctrl+S | Save output |
| F2 | Switch model |
| / | Toggle command palette |

## Status Bar

The status bar shows:
- Profile name
- Current model
- Token usage
- Tool count
- Gateway status
- Current time

## Command Palette

Press `/` to open the command palette. Type to filter commands. Use arrow keys to navigate and Enter to execute.
