# Media Reference

## Spotify

```bash
noman spotify play "song name"
noman spotify pause
noman spotify next
noman spotify search "artist"
noman spotify playlists
```

## YouTube

```bash
noman youtube search "topic"
noman youtube transcript --url https://youtube.com/watch?v=xxx
```

## GIF Search

```bash
noman gif search "keyword"
```
