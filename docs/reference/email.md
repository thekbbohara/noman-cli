# Email Reference

## Gmail

```bash
noman gmail list --max 10
noman gmail search "urgent"
noman gmail send --to user@example.com --subject "Hello" --body "Body text"
noman gmail delete <message_id>
```

## General Email

```bash
noman email list --max 10
noman email send --to user@example.com --subject "Hello" --body "Body"
noman email search "keyword"
```
