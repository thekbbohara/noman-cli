# Skills Hub Reference

## Overview

NoMan includes a skills hub for browsing, installing, searching, and publishing skills.

## CLI Commands

```bash
noman skill browse              # Browse available skills
noman skill search QUERY        # Search skills
noman skill install ID          # Install a skill
noman skill publish PATH        # Publish a skill
noman skill update              # Update outdated skills
noman skill list                # List installed skills
```

## Skill Registry

Skills can be sourced from:
- Local skills directory (`~/.noman/skills/`)
- Remote registry (built-in)
- GitHub repositories
- Custom registries
