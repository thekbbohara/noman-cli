# PDF Reference

## CLI Commands

```bash
noman pdf text --file document.pdf         # Extract text
noman pdf ocr --file scanned.pdf           # OCR for scanned PDFs
noman pdf merge --files a.pdf b.pdf -o out.pdf  # Merge PDFs
noman pdf split --file document.pdf        # Split PDF
noman pdf markdown --file document.pdf     # Convert to markdown
```

## Features

- Text extraction
- Image extraction
- Metadata extraction
- OCR for scanned documents
- PDF creation from text/markdown
- Merge and split
