## Codebase Patterns
- core/memory/store.py uses lazy connection pattern: _conn is None until first access
- core/memory/store.py: Use _require_conn() helper for safe database access
- cli/tui.py subclasses Textual's RichLog widget — overrides must match RichLog's signature
- Textual notify() severity literals: 'information', 'warning', 'error' (NOT 'info')
- Python 3.11+ StrEnum preferred over (str, Enum) tuple inheritance
- available_tools/system_prompt are @property methods in PromptAssembler, not class constants
- mypy strict mode: union-attr errors require explicit None checks before attribute access
- reactive() with lambda needs explicit TUIMetrics type annotation: reactive(lambda: TUIMetrics())
- Static widgets from self.query() need isinstance check before accessing renderable
